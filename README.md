# zowe-cli-cics-deploy-plugin

This project provides a plug-in for [Zowe CLI](https://github.com/zowe/zowe-cli) to deploy applications developed on a workstation to IBM速 CICS速 Transaction Server for z/OS速 (CICS). It aims to provide an experience similar to deploying to a cloud platform when deploying to CICS速.
